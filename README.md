# Divya Varma Chikitsai Maiyam (திவ்யா வர்ம சிகிச்சை மையம்)

A modern, responsive healthcare and traditional Varma therapy website for **Divya Varma Chikitsai Maiyam** & **Divya Allied Health Science And Technology**, Sivakasi, Tamil Nadu.

## 🌿 Features

- **2-Column Hero Section**: Modern UI with a circular glowing clinic logo on left and clinic info on right.
- **Dr. Shanthalakshmi M.A., MD (Varma Therapy)** credentials & specialized treatments showcase.
- **Services Offered**: Varma Therapy, Acupuncture, Cupping/Hijama, Panchakarma, Herbal Medicine.
- **Courses & Training**: Allied Health Science training course details and admissions info.
- **Appointment Contact Form**:
  - Direct email notification sent to `divyavarmacentre@gmail.com` (via FormSubmit & SMTP).
  - Saved to Firebase Firestore database (`appointments` collection).
- **Mobile Responsive Drawer Menu**: Glassmorphism navbar with smooth section scrolling.

## 🚀 How to Run Locally

```bash
# Clone the repository
git clone https://github.com/SivabalanManogar/Divya-Clinic-.git

# Navigate to project folder
cd Divya-Clinic-

# Start local server
npx serve -l 3000
```

Open `http://localhost:3000` in your web browser.
